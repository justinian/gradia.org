#  Perform 
**Ability**: Charisma
* *Specialized**

The Perform skill represents a character's ability to entertain an audience. A Perform skill must have an associated specialization, representing the type of performance. This skill can be selected multiple times, with each specialization being a separate skill.

Example Specializations:
* Acting
* Comedy
* Dance
* Keyboard Instruments
* Oratory
* Percussion Instruments
* Song
* Stringed Instruments
* Wind Instruments
