# Gradia Sourcebook

This is the source for the static sourcebook-site for our custom fantasy
setting, [Gradia][]. The content for this site originally came from a [MoinMoin][]
wiki, which we're in the process of cleaning up into a static site.

[Gradia]: http://gradia.org
[MoinMoin]: http://moinmo.in

This site uses [Hugo][] for generation, and is based off of a [Bootstrap][]
theme for Hugo, [Beg][].

[Hugo]: http://gohugo.io
[Bootstrap]: http://getbootstrap.com
[Beg]: https://themes.gohugo.io/beg/
